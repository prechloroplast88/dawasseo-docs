# 다왔어(DaWasseo) 개인정보처리방침

**시행일**: 2026-05-30
**최종 개정일**: 2026-05-30

다왔어(이하 "본 앱")는 대중교통 목적지 도착 알림 서비스를 제공하며, 이용자의 개인정보를 다음과 같이 처리합니다.

---

## 1. 수집·처리하는 개인정보 항목

본 앱은 서비스 제공에 필요한 최소한의 정보만 처리합니다.

| 구분 | 항목 | 수집 방법 |
| --- | --- | --- |
| 위치 정보 | 단말기의 GPS 좌표(위·경도), 이동 속도, 측위 정확도 | 사용자가 위치 권한을 허용한 후 단말기에서 자동 수집 |
| 알람 설정 | 목적지 좌표·이름, 알람 반경, 즐겨찾기 목록, 알림 유형 설정 | 사용자가 직접 입력 |
| 운영 로그 | 알람 시작/종료 시각, 위치 추적 이벤트, 오류 메시지 (베타 한정, 아래 4항 참고) | 앱 내부에서 자동 생성 |

본 앱은 이름·전화번호·이메일·주소 등 **개인 식별 정보를 수집하지 않습니다.**

## 2. 개인정보 처리 목적

수집한 정보는 아래 목적으로만 사용됩니다.

1. 사용자가 설정한 목적지에 도착했을 때 알림 제공
2. 지하 등 GPS 신호가 약한 구간에서 지하철 역 데이터베이스 기반 위치 추정
3. 알람 설정·즐겨찾기 저장 및 복원
4. **(베타 기간 한정)** 백그라운드 위치 추적·알람 발동 안정성 디버깅

## 3. 보유 및 이용 기간

- **위치 정보**: 알람이 진행되는 동안에만 메모리상에서 사용되며, 알람이 종료되면 단말기에서 즉시 폐기됩니다. 별도의 서버에 영구 저장하지 않습니다.
- **알람 설정·즐겨찾기**: 사용자의 단말기 내부 저장소(SharedPreferences)에만 보관되며, 앱을 삭제하면 함께 삭제됩니다.
- **운영 로그**: 단말기 로컬에 임시 저장되며 알람 종료 시 또는 사용자가 앱을 삭제할 때 폐기됩니다.

## 4. 베타 기간 디버깅 로그 전송에 관한 고지

본 앱은 **클로즈드 베타 기간 동안** 백그라운드 동작 안정성 검증을 위해 운영 로그를 개발자가 운영하는 Discord 채널로 자동 전송합니다.

- **전송 항목**: 시각, 알람 단계 이벤트, 대략적 위치(위·경도), 측위 정확도, 오류 메시지
- **전송되지 않는 항목**: 단말기 식별자, 사용자 이름·이메일, 다른 앱 정보, 사진·연락처 등
- **수신처**: 개발자가 단독으로 접근 가능한 비공개 Discord 채널
- **이용 목적**: 알람 미발동·오발동 사례 분석 및 수정에 한정
- **opt-out**: 앱 설정 화면의 "개발자에게 로그 자동 전송" 스위치로 언제든 끌 수 있습니다.
- **종료 시점**: **정식 출시 빌드에서는 본 기능이 완전히 제거**되며, 이 단락도 함께 삭제됩니다.

## 5. 제3자 제공 및 처리 위탁

본 앱은 수집한 위치 정보를 광고·마케팅·통계 등 어떠한 목적으로도 **제3자에게 제공하지 않습니다.** 베타 기간 Discord 전송은 위 4항에 명시된 디버깅 목적 외의 용도로 사용되지 않습니다.

본 앱은 지도 타일(OpenStreetMap), 장소 검색(Kakao Local API / OpenStreetMap Nominatim) 서비스를 이용하며, 검색어와 화면에 표시할 좌표 범위가 해당 서비스 제공자에게 전달됩니다.

## 6. 자동 수집 정보 및 거부 방법

본 앱은 다음 항목을 **수집하지 않습니다.**

- HTTP 쿠키 (본 앱은 웹 브라우저가 아닙니다)
- 광고 식별자(IDFA, AAID 등)
- 단말기 고유 식별자(IMEI, UUID 등)
- 사용자 행동 분석을 위한 별도 SDK(Google Analytics, Firebase Analytics, Crashlytics 등)

지도·장소 검색 기능을 위해 OpenStreetMap 및 Kakao Local API 서버에 요청을 보내며, 이 과정에서 해당 서비스 제공자의 표준 로그(IP 주소, User-Agent 등)가 일시적으로 기록될 수 있습니다. 이는 각 서비스 제공자의 자체 정책에 따라 처리됩니다.

OS 단의 자동 수집(예: 시스템 진단 데이터)은 단말기 설정의 개인정보·진단 메뉴에서 제어할 수 있습니다.

## 7. 이용자의 권리 및 행사 방법

- **위치 권한 철회**: 단말기 설정 → 앱 → 다왔어 → 권한에서 위치 권한을 언제든 해제할 수 있습니다. 권한이 해제되면 알람 기능이 동작하지 않습니다.
- **저장 데이터 삭제**: 앱을 삭제하면 단말기에 저장된 모든 알람 설정·즐겨찾기·로그가 함께 삭제됩니다.
- **Discord 로그 전송 거부**: 앱 설정 화면의 "개발자에게 로그 자동 전송" 스위치를 꺼서 언제든 비활성화할 수 있습니다.
