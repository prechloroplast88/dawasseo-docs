# dawasseo-docs

다왔어(DaWasseo) 앱의 공개 문서 호스팅 저장소.

본 저장소는 앱 본체 소스 코드와 분리된 **개인정보처리방침 등 공개가 필요한 문서만** 호스팅하기 위해 존재합니다.

## 호스팅 URL

GitHub Pages 활성화 후:

- 메인: `https://prechloroplast88.github.io/dawasseo-docs/`
- 개인정보처리방침: `https://prechloroplast88.github.io/dawasseo-docs/privacy_policy.html`

## 갱신 절차

처리방침 본문은 앱 본체 저장소(`dawasseo`)의 `docs/privacy_policy.md`가 작업 기준입니다. 변경 시 본 저장소의 `privacy_policy.md`도 동일하게 갱신해주세요.
